Javascript Variables

What is a Variable?

A variable in JavaScript is like a storage box where you can keep a value that you can use later. Think of it like naming a box and then putting something inside it.

var myName = "John";

Parts of a Variable in JavaScript

A variable in JavaScript is made up of three main parts:

  1. Declaration Keyword:

    This is the keyword used to declare a variable. The common keywords are var, let, and const.

    • var: The old way of declaring variables, used globally or function-scoped.
    • let: The modern way to declare block-scoped variables.
    • const: Used to declare block-scoped constants that cannot be reassigned.
  2. Variable Name (Identifier):

    This is the name you give to the variable so you can reference it later. The name should follow these rules:

    • Must start with a letter, underscore (_), or dollar sign ($).
    • Can contain letters, numbers, underscores, or dollar signs after the first character.
    • Case-sensitive (e.g., myVariable and myvariable are different).

    Example: let age;

  3. Assignment Operator and Value:

    This is the equal sign (=) used to assign a value to the variable. The value can be any data type, such as a number, string, boolean, etc.

    Example: let age = 16;

Example Code

Here's an example of a variable declaration in JavaScript:

let name = "Alice";

In this example:

  • let is the declaration keyword.
  • name is the variable name.
  • = "Alice" is the assignment operator and value.

Choosing Between let and const

  • Use let if you expect the value of the variable to change.
  • Use const if the value should stay the same.

Variable Names

When naming a variable, follow these rules:

  • Start with a letter, underscore (_), or dollar sign ($).
  • After the first character, you can use numbers, letters, underscores, or dollar signs.
  • Variables are case-sensitive, so myVariable and myvariable are different.
